# Daniel Smidstrup — Personal Page

One-page, SEO-friendly, ultra-fast personal site built with Astro + Tailwind, with a tiny React “scroll accent” island.

## Local dev

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Replace images (headshot + social preview)

- **Headshot**: drop one of these into `public/`:
  - `public/headshot.jpg` (preferred)
  - `public/headshot.png`

The page will fall back to `public/headshot.svg` if you haven’t added a real photo yet.

- **Social preview image**: currently uses `public/og.svg`.
  - If you prefer a PNG, add `public/og.png` and update the `ogImageUrl` in `src/layouts/Base.astro`.

## SEO / canonical URL / sitemap

- Sitemap is generated by `@astrojs/sitemap` on build.
- Set a real site URL for canonical links + sitemap:
  - In Vercel, add an env var named `SITE` (example: `https://daniel.com`).
  - This maps to `site` in `astro.config.mjs`.

## Deploy to Vercel (recommended)

1. Push this folder to GitHub.
2. In Vercel: **New Project → Import**.
3. Framework: Astro (auto-detected).
4. Build Command: `npm run build`
5. Output Directory: `dist`
6. Add env var `SITE` (your custom domain) once you have it.
